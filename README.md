Next.jsで実装したフロントエンドのみで動作するWebアプリケーション集

https://nakamura0907.github.io/front-only-apps/

## 主な使用技術

- TypeScript: 5.0
- Next.js: 14.0
- GitHub Actions

### ディレクトリ構成

```bash
./src
├── app
├── components
├── config # 定数ほか
├── hooks
├── providers
└── vendor # 上記に該当しない外部ライブラリ
```
